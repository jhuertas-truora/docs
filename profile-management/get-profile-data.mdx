## Overview

Use this guide to retrieve profile data by unique identifier (for example, `profile_id` or `email`).

<Steps>
<Step title="Prerequisites">
  - You have a valid API key
  - You know the profile identifier (e.g., `profile_id` or `email`)
</Step>

<Step title="Make the request">
  <RequestExample>
  ```bash cURL
  curl -X GET "https://api.example.com/profiles/{profile_id}" \
    -H "Authorization: Bearer $API_KEY"
  ```
  </RequestExample>

  <ResponseExample>
  ```json Success
  {
    "id": "prof_123",
    "email": "user@example.com",
    "created_at": "2024-01-15T10:30:00Z",
    "variables": {
      "first_name": "Alex",
      "plan": "pro"
    },
    "tags": ["beta", "customer"]
  }
  ```
  </ResponseExample>
</Step>

<Step title="Understand the response">
  <ResponseField name="id" type="string" required>
  Unique profile identifier.
  </ResponseField>
  <ResponseField name="email" type="string">
  Primary email address.
  </ResponseField>
  <ResponseField name="variables" type="object">
  Custom key-value attributes associated to the profile.
  </ResponseField>
  <ResponseField name="tags" type="array">
  Labels used for organization and segmentation.
  </ResponseField>
</Step>
</Steps>

<Tip>
Prefer the immutable `id` over email when available to avoid lookup ambiguity.
</Tip>
