---
title: "Remove tags from a profile"
description: "Detach one or more tags from a profile."
---

## Overview

Remove tags when they no longer apply.

<Steps>
<Step title="Remove specific tags">
  <RequestExample>
  ```bash cURL
  curl -X DELETE "https://api.example.com/profiles/{profile_id}/tags" \
    -H "Authorization: Bearer $API_KEY" \
    -H "Content-Type: application/json" \
    -d '{"tags": ["beta"]}'
  ```
  </RequestExample>
</Step>

<Step title="Remove all tags">
  <RequestExample>
  ```bash cURL
  curl -X DELETE "https://api.example.com/profiles/{profile_id}/tags/all" \
    -H "Authorization: Bearer $API_KEY"
  ```
  </RequestExample>
</Step>
</Steps>

<Info>
Keep audit logs when modifying tags for compliance and traceability.
</Info>
