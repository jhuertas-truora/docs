Uma transação representa um registro de uma ação realizada por um usuário através dos produtos da Truora. Cada transação captura informações relevantes que contribuem para construir ou atualizar um perfil de usuário. As transações servem como a unidade de dados central para tomada de decisões, avaliação de risco e análise do comportamento do usuário.

## Pontos Principais:

### Fonte de Dados
As transações são geradas sempre que um usuário interage com um produto ou serviço da Truora.

### Relação com Perfis
Cada transação está vinculada a um perfil de usuário usando identificadores (ID Rígido, ID Flexível, ou ID Interno), garantindo que todas as ações estejam associadas ao indivíduo correto.

Produtos da Truora que Geram Transações

- Verificação de Identidade
- Verificação Facial
- Verificações de Antecedentes
- Formulários
- Chats do WhatsApp
- Assinatura Eletrônica
- Verificação Telefônica
- Verificação de Email
