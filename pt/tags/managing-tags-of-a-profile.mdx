## Atribuir tags a um perfil

Atribuir tags é a forma principal de segmentar e classificar usuários. Esta seção explica como ver as tags atuais em um perfil e como adicioná-las ou removê-las.

### Ver Tags Existentes em um Perfil

Para entender como um usuário está atualmente segmentado, você pode encontrar todas as suas tags atribuídas diretamente em seu perfil. Esta informação está localizada centralmente na visualização do perfil do contato.

Navegue até o perfil do usuário e localize a seção **"Informações de contato"**. Dentro desta seção, você verá o campo **"Tags"**.

![Visualização do perfil do usuário mostrando a lista de tags atribuídas](../../images/tagsGuide/profile.png)

* **Lista de Tags:** Esta área exibe todas as tags associadas ao usuário como "pílulas" visuais.
* **Nome e Cor:** O texto em cada pílula corresponde à propriedade `name` da tag (ex., "Hot", "Truorateam"), e a cor de fundo é determinada por sua propriedade `color`.
* **Ponto de Partida:** Para modificar essas tags, clique no link **"Gerenciar tags"** localizado logo abaixo da lista.

### Adicionar e Remover Tags

Após clicar em "Gerenciar tags", uma janela modal será aberta. Esta interface é o centro de controle para modificar as tags de um usuário específico. A visualização principal é a aba **"Editar tags"**.

![Janela modal para editar tags do usuário](../../images/tagsGuide/editTag.png)

* **Atribuir uma Tag:** Para adicionar uma tag, clique no campo de entrada **"Tags do usuário"**. Uma lista pesquisável de todas as tags disponíveis será exibida. Selecione uma ou mais tags para atribuí-las ao perfil.
* **Remover uma Tag:** As tags que um usuário já possui aparecem como pílulas. Para remover uma, simplesmente clique no ícone **(x)** localizado à direita do nome da tag.
* **Salvar Alterações:** Uma vez que você terminou de editar, é **essencial** clicar no botão **"Salvar"** para aplicar todas as modificações. Se você fechar a janela sem salvar, todas as alterações serão perdidas.
