---
title: "Remover tags de um perfil"
description: "Desconecte uma ou mais tags de um perfil."
---

## Resumo

Remova tags quando não se aplicarem mais.

<Steps>
<Step title="Remover tags específicas">
  <RequestExample>
  ```bash cURL
  curl -X DELETE "https://api.example.com/profiles/{profile_id}/tags" \
    -H "Authorization: Bearer $API_KEY" \
    -H "Content-Type: application/json" \
    -d '{"tags": ["beta"]}'
  ```
  </RequestExample>
</Step>

<Step title="Remover todas as tags">
  <RequestExample>
  ```bash cURL
  curl -X DELETE "https://api.example.com/profiles/{profile_id}/tags/all" \
    -H "Authorization: Bearer $API_KEY"
  ```
  </RequestExample>
</Step>
</Steps>

<Info>
Mantenha registros de auditoria ao modificar tags para conformidade e rastreabilidade.
</Info>
