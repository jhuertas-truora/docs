## Visão Geral

Uma Plataforma de Dados de Clientes (CDP) é um sistema de software que coleta, unifica e organiza dados de clientes de múltiplas fontes de produtos da Truora em um perfil único, consistente e abrangente para cada cliente. Diferente de armazéns de dados tradicionais ou CRMs, uma CDP é projetada para ser acessível tanto para equipes técnicas quanto não técnicas, permitindo segmentação em tempo real, personalização e ativação de dados através de canais.

## Descrição da API

A API de CDP fornece acesso seguro e centralizado aos dados de clientes armazenados na sua Plataforma de Dados de Clientes. Ela permite que você:
- Recupere e gerencie perfis de clientes, incluindo atributos pessoais e variáveis de decisão.
- Acesse e analise transações associadas a cada perfil.
- Marque e categorize perfis com rótulos personalizados para melhorar a segmentação.
- Crie e gerencie segmentos dinâmicos de clientes para ações direcionadas e insights.

Esta API é projetada para ajudá-lo a unificar informações do cliente, otimizar a tomada de decisões baseada em dados e ativar insights em todo o ecossistema da sua organização.

- Os benefícios principais incluem:
- Acesso centralizado a dados para uma visão completa do cliente 360°.
- Segmentação e marcação em tempo real para gerenciamento flexível de audiências.
- Endpoints REST seguros e escaláveis com respostas JSON consistentes.
- Design pronto para integração com plataformas de marketing, CRMs e ferramentas de análise.

## Conceitos Centrais

Uma Plataforma de Dados de Clientes (CDP) é construída sobre um conjunto de conceitos centrais que definem como a informação do cliente é capturada, organizada e ativada. Entender esses conceitos é essencial para aproveitar ao máximo a API.

### Perfis

Um perfil representa uma identidade única de cliente na CDP. Ele consolida atributos como nome, email, informações de documentos e sinais comportamentais. Os perfis são persistentes e servem como a base de todas as interações com clientes.

### Transações

Uma transação captura um evento ou interação associada a um cliente. As transações podem incluir compras, validações ou outras atividades relevantes para o seu negócio. Elas são marcadas com tempo e vinculadas a perfis, fornecendo uma visão histórica do comportamento do cliente e mudanças de status.

### Variáveis de Decisão

As variáveis de decisão são atributos dinâmicos armazenados dentro de um perfil que influenciam como um cliente é segmentado ou engajado.

### Tags

As tags permitem que você categorize e rotule perfis baseados em critérios personalizados, elas podem representar segmentos como "Cliente VIP", "Em Risco", ou "Nível de Engajamento: Alto".

As tags facilitam a filtragem e agrupamento de perfis.

### Segmentos

Um segmento é uma coleção dinâmica de perfis que compartilham atributos comuns, variáveis de decisão ou tags. Os segmentos são baseados em regras e atualizados continuamente, tornando-os essenciais para personalização, direcionamento e análise. Eles também podem ser usados para acionar ações ou visualizar dados efetivamente.
