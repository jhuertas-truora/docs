## Resumo

Use este guia para recuperar dados de perfil por identificador único (por exemplo, `profile_id` ou `email`).

<Steps>
<Step title="Pré-requisitos">
  - Você tem uma chave API válida
  - Você conhece o identificador do perfil (ex., `profile_id` ou `email`)
</Step>

<Step title="Fazer a solicitação">
  <RequestExample>
  ```bash cURL
  curl -X GET "https://api.example.com/profiles/{profile_id}" \
    -H "Authorization: Bearer $API_KEY"
  ```
  </RequestExample>

  <ResponseExample>
  ```json Success
  {
    "id": "prof_123",
    "email": "user@example.com",
    "created_at": "2024-01-15T10:30:00Z",
    "variables": {
      "first_name": "Alex",
      "plan": "pro"
    },
    "tags": ["beta", "customer"]
  }
  ```
  </ResponseExample>
</Step>

<Step title="Entender a resposta">
  <ResponseField name="id" type="string" required>
  Identificador único do perfil.
  </ResponseField>
  <ResponseField name="email" type="string">
  Endereço de email principal.
  </ResponseField>
  <ResponseField name="variables" type="object">
  Atributos chave-valor personalizados associados ao perfil.
  </ResponseField>
  <ResponseField name="tags" type="array">
  Etiquetas usadas para organização e segmentação.
  </ResponseField>
</Step>
</Steps>

<Tip>
Prefira o `id` imutável sobre o email quando disponível para evitar ambiguidade na busca.
</Tip>
