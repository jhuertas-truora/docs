Una transacción representa un registro de una acción realizada por un usuario a través de los productos de Truora. Cada transacción captura información relevante que contribuye a construir o actualizar un perfil de usuario. Las transacciones sirven como la unidad de datos central para la toma de decisiones, evaluación de riesgo y análisis del comportamiento del usuario.

## Puntos Clave:

### Fuente de Datos
Las transacciones se generan cada vez que un usuario interactúa con un producto o servicio de Truora.

### Relación con Perfiles
Cada transacción está vinculada a un perfil de usuario usando identificadores (ID Duro, ID Blando, o ID Interno), asegurando que todas las acciones estén asociadas con el individuo correcto.

Productos de Truora que Generan Transacciones

- Verificación de Identidad
- Verificación Facial
- Verificaciones de Antecedentes
- Formularios
- Chats de WhatsApp
- Firma Electrónica
- Verificación Telefónica
- Verificación de Email
