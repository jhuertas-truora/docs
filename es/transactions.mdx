## Comienza en tres pasos

Pon tu sitio de documentación funcionando localmente y haz tu primera personalización.

### Paso 1: Configura tu entorno local

<AccordionGroup>
  <Accordion icon="copy" title="Clona tu documentación localmente">
    Durante el proceso de incorporación, creaste un repositorio de GitHub con el contenido de tu documentación si no tenías uno ya. Puedes encontrar un enlace a este repositorio en tu [panel de control](https://dashboard.mintlify.com).
    
    Para clonar el repositorio localmente para que puedas hacer y previsualizar cambios en tu documentación, sigue la guía [Clonar un repositorio](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) en la documentación de GitHub.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Inicia el servidor de previsualización">
    1. Instala la CLI de Mintlify: `npm i -g mint`
    2. Navega a tu directorio de documentación y ejecuta: `mint dev`
    3. Abre `http://localhost:3000` para ver tu documentación en vivo!
    
    <Tip>Tu previsualización se actualiza automáticamente mientras editas archivos.</Tip>
  </Accordion>
</AccordionGroup>

### Paso 2: Despliega tus cambios

<AccordionGroup>
  <Accordion icon="github" title="Instala nuestra aplicación de GitHub">
    Instala la aplicación de GitHub de Mintlify desde tu [panel de control](https://dashboard.mintlify.com/settings/organization/github-app).
    
    Nuestra aplicación de GitHub despliega automáticamente tus cambios a tu sitio de documentación, por lo que no necesitas gestionar despliegues tú mismo.
</Accordion>
<Accordion icon="palette" title="Actualiza el nombre y colores de tu sitio">
    Para un primer cambio, actualicemos el nombre y colores de tu sitio de documentación.

    1. Abre `docs.json` en tu editor.
    2. Cambia el campo `"name"` al nombre de tu proyecto.
    3. Actualiza los `"colors"` para que coincidan con tu marca.
    4. Guarda y ve tus cambios instantáneamente en `http://localhost:3000`.

    <Tip>¡Prueba cambiar el color primario para ver una diferencia inmediata!</Tip>
  </Accordion>
</AccordionGroup>

### Paso 3: Sal a la luz

<Accordion icon="rocket" title="Publica tu documentación">
  1. Confirma y envía tus cambios.
  2. ¡Tu documentación se actualizará y estará en vivo en momentos!
</Accordion>
