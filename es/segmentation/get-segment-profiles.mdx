title: "Obtener perfiles en un segmento"
description: "Lista perfiles que actualmente coinciden con un segmento guardado."
---

## Resumen

Obtén miembros actuales de un segmento con paginación.

<Steps>
<Step title="Consultar un segmento">
  <RequestExample>
  ```bash cURL
  curl -G "https://api.example.com/segments/{segment_id}/profiles" \
    -H "Authorization: Bearer $API_KEY" \
    --data-urlencode "limit=50"
  ```
  </RequestExample>
</Step>

<Step title="Campos de respuesta">
  <ResponseField name="data" type="array" required>
  Lista de objetos de perfil.
  </ResponseField>
  <ResponseField name="next_cursor" type="string">
  Cursor para obtener la siguiente página.
  </ResponseField>
</Step>
</Steps>
