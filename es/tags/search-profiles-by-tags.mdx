## Resumen

Consulta perfiles que tengan etiquetas específicas adjuntas.

<Steps>
<Step title="Filtrar por una o múltiples etiquetas">
  <RequestExample>
  ```bash cURL
  curl -G "https://api.example.com/profiles/search" \
    -H "Authorization: Bearer $API_KEY" \
    --data-urlencode "tags=beta,vip" \
    --data-urlencode "logic=and"
  ```
  </RequestExample>
</Step>

<Step title="Paginación y ordenamiento">
  - Usa `limit` y `cursor` para paginación
  - Ordena por `last_seen_at` o `created_at`
</Step>
</Steps>
