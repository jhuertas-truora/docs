## Resumen

Usa esta guía para recuperar datos de perfil por identificador único (por ejemplo, `profile_id` o `email`).

<Steps>
<Step title="Prerrequisitos">
  - Tienes una clave API válida
  - Conoces el identificador del perfil (ej., `profile_id` o `email`)
</Step>

<Step title="Hacer la solicitud">
  <RequestExample>
  ```bash cURL
  curl -X GET "https://api.example.com/profiles/{profile_id}" \
    -H "Authorization: Bearer $API_KEY"
  ```
  </RequestExample>

  <ResponseExample>
  ```json Success
  {
    "id": "prof_123",
    "email": "user@example.com",
    "created_at": "2024-01-15T10:30:00Z",
    "variables": {
      "first_name": "Alex",
      "plan": "pro"
    },
    "tags": ["beta", "customer"]
  }
  ```
  </ResponseExample>
</Step>

<Step title="Entender la respuesta">
  <ResponseField name="id" type="string" required>
  Identificador único del perfil.
  </ResponseField>
  <ResponseField name="email" type="string">
  Dirección de correo electrónico principal.
  </ResponseField>
  <ResponseField name="variables" type="object">
  Atributos clave-valor personalizados asociados al perfil.
  </ResponseField>
  <ResponseField name="tags" type="array">
  Etiquetas utilizadas para organización y segmentación.
  </ResponseField>
</Step>
</Steps>

<Tip>
Prefiere el `id` inmutable sobre el email cuando esté disponible para evitar ambigüedad en la búsqueda.
</Tip>
