<Info>
  **Prerrequisitos**:
  - Node.js versión 19 o superior
  - Un repositorio de documentación con un archivo `docs.json`
</Info>

Sigue estos pasos para instalar y ejecutar Mintlify en tu sistema operativo.

<Steps>
<Step title="Instalar la CLI de Mintlify">

```bash
npm i -g mint
```
</Step>

<Step title="Previsualizar localmente">

Navega a tu directorio de documentación donde se encuentra tu archivo `docs.json`, y ejecuta el siguiente comando:

```bash
mint dev
```

Una previsualización local de tu documentación estará disponible en `http://localhost:3000`.

</Step>
</Steps>

## Puertos personalizados

Por defecto, Mintlify usa el puerto 3000. Puedes personalizar el puerto en el que ejecuta Mintlify usando la bandera `--port`. Por ejemplo, para ejecutar Mintlify en el puerto 3333, usa este comando:

```bash
mint dev --port 3333
```

Si intentas ejecutar Mintlify en un puerto que ya está en uso, usará el siguiente puerto disponible:

```md
El puerto 3000 ya está en uso. Intentando 3001 en su lugar.
```

## Versiones de Mintlify

Ten en cuenta que cada lanzamiento de CLI está asociado con una versión específica de Mintlify. Si tu previsualización local no coincide con la versión de producción, por favor actualiza la CLI:

```bash
npm mint update
```

## Validando enlaces

La CLI puede ayudar con la validación de enlaces en tu documentación. Para identificar enlaces rotos, usa el siguiente comando:

```bash
mint broken-links
```

## Formato de código

Sugerimos usar extensiones en tu IDE para reconocer y formatear MDX. Si eres usuario de VSCode, considera la [extensión MDX VSCode](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) para resaltado de sintaxis, y [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) para formateo de código.

## Solución de problemas

<AccordionGroup>
  <Accordion title='Error: No se pudo cargar el módulo "sharp" usando el runtime darwin-arm64'>

    Esto puede deberse a una versión desactualizada de node. Prueba lo siguiente:
    1. Elimina la versión actualmente instalada de la CLI: `npm remove -g mint`
    2. Actualiza a Node v19 o superior.
    3. Reinstala la CLI: `npm i -g mint`
  </Accordion>

  <Accordion title="Problema: Encontrando un error desconocido">
  
    Solución: Ve a la raíz de tu dispositivo y elimina la carpeta `~/.mintlify`. Luego ejecuta `mint dev` nuevamente.
  </Accordion>
</AccordionGroup>

¿Curioso sobre qué cambió en la última versión de CLI? Consulta el [registro de cambios de CLI](https://www.npmjs.com/package/mintlify?activeTab=versions).
