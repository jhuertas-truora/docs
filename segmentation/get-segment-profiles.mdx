---

## Overview

Fetch current members of a segment with pagination.

<Steps>
<Step title="Query a segment">
  <RequestExample>
  ```bash cURL
  curl -G "https://api.example.com/segments/{segment_id}/profiles" \
    -H "Authorization: Bearer $API_KEY" \
    --data-urlencode "limit=50"
  ```
  </RequestExample>
</Step>

<Step title="Response fields">
  <ResponseField name="data" type="array" required>
  List of profile objects.
  </ResponseField>
  <ResponseField name="next_cursor" type="string">
  Cursor for fetching the next page.
  </ResponseField>
</Step>
</Steps>
